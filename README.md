
# Python Simple PubSub

A simple message pub/sub by tornado, rabbitmq and websocket

## Prerequisite

1. python2.7+
2. python-pip
3. rabbitmq-server

## Installation

```
$ pip install -r requirements
```

## Run Server

```
$ python server.py
```

## Connect

Open a browser and input the following address: *http://localhost:9999/*
